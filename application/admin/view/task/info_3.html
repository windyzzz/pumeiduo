操作提示

  • 新增任务逻辑 —— 累加（每次的情况下）
    例如：用户的第一个设置任务A已完成1人（1次机会），新增第二个设置任务B为邀请1人，此时设置任务B未完成（需要后续有完成任务才能更新）
    再完成1人（实际2人），设置任务A会变成 2（2次机会），设置任务B会变成 0 + 2 = 2（2次机会）
    再完成1人（实际3人），设置任务A会变成 3（3次机会），设置任务B会变成 2 + 1 = 3（3次机会）
    再完成1人（实际4人），设置任务A会变成 4（4次机会），设置任务B会变成 3 + 1 = 4（4次机会）
